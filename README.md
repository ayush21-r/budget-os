# BudgetOS

BudgetOS is a frontend-only personal budgeting dashboard built with React, Vite, JavaScript, CSS Modules, Recharts, and Lucide React.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

This phase uses React state and Local Storage only. It does not include authentication, a backend, a database, or API calls.
