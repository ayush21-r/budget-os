# BudgetOS

BudgetOS is a frontend-only personal budgeting dashboard built with React, Vite, JavaScript, CSS Modules, Recharts, and Lucide React.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Budget and settings data are persisted in Supabase. Apply `supabase-schema.sql` to the Supabase project before running the app.
